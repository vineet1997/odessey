## Global Rules

### Color Palette ‚Äî "wine-dark sea" (dark mode ONLY)

| Role | Hex | CSS Variable |
|------|-----|--------------|
| Background | `#0A0E14` | `--bg` (Aegean night ‚Äî never pure #000) |
| Raised surface | `#111722` | `--bg-raised` (cards, decks, sheets) |
| Ink | `#E9E4D8` | `--ink` (warm foam white ‚Äî never #FFF) |
| Ink muted | `#8B94A3` | `--ink-muted` |
| Gold | `#C9A227` | `--gold` (verdicts, CTAs, wordmark, focus rings ‚Äî large text/surfaces) |
| Gold bright | `#E3C158` | `--gold-bright` (small text & links, ‚â•4.5:1 on `--bg`) |
| Sea | `#2E6E73` | `--sea` (transport/metro surfaces, secondary actions) |
| Sea bright | `#5FA8AD` | `--sea-bright` (transport text on dark) |
| Wine | `#722F37` | `--wine` (no-way-home warning surface ONLY) |
| Wine bright | `#C96F6F` | `--wine-bright` (warning text) |
| Border | `rgba(233,228,216,0.10)` | `--border` (hairlines) |

**Semantic law:** gold = verdict/action ¬∑ sea = journey ¬∑ wine = stranded. No decorative use of these three; color always carries meaning. Metro chips use real DMRC line colors (Yellow `#FFD200`, Magenta `#B5379A`, Blue `#3B8ACD`‚Ä¶) with dark text, small sizes only.

### Typography ‚Äî triple stack, NO UI sans anywhere

- **Display:** Cinzel 500/600 ‚Äî wordmark, venue names, intent titles. Sparingly.
- **Body:** Source Serif 4 300‚Äì600 ‚Äî editorial sentences; italic for one-line venue verdicts.
- **Data:** JetBrains Mono 400/500 ‚Äî times, prices, distances, freshness stamps, labels; uppercase, tracking-widest, 12‚Äì13px.
- **Voice rule:** opinions in serif, evidence in mono.

**Google Fonts:**
```css
@import url('https://fonts.googleapis.com/css2?family=Cinzel:wght@500;600&family=Source+Serif+4:ital,wght@0,300;0,400;0,600;1,400&family=JetBrains+Mono:wght@400;500&display=swap');
```

**Tailwind:** `fontFamily: { display: ['Cinzel','serif'], body: ['"Source Serif 4"','serif'], mono: ['"JetBrains Mono"','monospace'] }`

**Scale (mobile):** mono labels 12‚Äì13px ¬∑ serif body 16px/1.6 ¬∑ venue name 28‚Äì32px Cinzel ¬∑ showtime hero 40px mono ¬∑ prologue statements `clamp(40px, 10vw, 72px)`.

### Shadows

Dark-on-dark: shadows are near-invisible ‚Äî depth comes from `--bg` vs `--bg-raised` contrast + hairline borders + the gold hairline draw on the result card. Use `0 12px 32px rgba(0,0,0,0.45)` only for the sheet/modal layer.

## Motion

Ease: `expo.out` ‚Äî `cubic-bezier(0.16,1,0.3,1)` everywhere. Durations: 150 micro / 300 transitions / 500 arrivals. Press scale 0.97. Hover (desktop) y ‚àí4px scale 1.02 with paired reverse tween. Stagger 0.03s. `prefers-reduced-motion`: transforms ‚Üí 150ms fades; prologue ‚Üí static pages.

**Prologue scrub (the app's ONLY pinned section):**
```js
gsap.timeline({ scrollTrigger: { trigger: section, start: 'top top', end: '+=150%', scrub: 1, pin: true } })
  .from('.statement', { opacity: 0, y: 40 })
  .to('.bg-layer', { yPercent: -20 }, '<');
```
`ScrollTrigger.refresh()` after Cinzel loads (font swap shifts metrics). **No SplitText** (paid plugin) ‚Äî statements animate as whole blocks.

### Page Pattern

**Voyage flow:** 1. Prologue (pinned scrub, skippable) ‚Üí 2. The Helm (horizontal input deck, constellation progress) ‚Üí 3. The Crossing (route-draw loader, real latency, cap 1.5s) ‚Üí 4. Ithaka (result card + collapsed runner-up + intent tabs). CTA: sticky bottom action row, safe-area padded.

## Anti-Patterns (Do NOT Use)

- ‚ùå Inter or any UI sans ¬∑ ‚ùå pink/`#EC4899` (pre-revision artifact) ¬∑ ‚ùå shadcn default look ¬∑ ‚ùå glassmorphism blur soup ¬∑ ‚ùå pure `#000000` or `#FFFFFF`
- ‚ùå Spinners or skeleton shimmer ‚Äî the Crossing is the only loading state
- ‚ùå Greek ornament beyond the single hairline key divider per screen
- ‚ùå Emojis as icons ‚Äî Lucide 1.5px stroke + 3 custom glyphs (star-fix, metro chip, screen)
- ‚ùå Layout-shifting hovers ¬∑ ‚ùå instant state changes ¬∑ ‚ùå invisible focus states ¬∑ ‚ùå decorative use of gold/sea/wine
